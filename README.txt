This package is currently being developed.

New functions are being developed and a user guide is coming soon!

To install domo:

    1) Install devtools if you do not have it.

    2) With devtools installed, you can then install domo by executing the following: devtools::install_github("npm27/domo")

    3) Load the library to get started! library(domo)

    4) Enjoy screening data with domo!
